# FlexiPredict: AI-Driven Financial Intelligence Terminal

FlexiPredict is a high-fidelity financial dashboard designed for institutional-grade market analysis. It combines real-time price simulation with Google's Gemini 3 Pro AI to provide actionable trading insights for Gold, Silver, and major stocks.

- **Real-Time Market Engine**: Simulates live price fluctuations and volume changes every 5 seconds.

## üß† AI Analysis Logic (`services/gemini.ts`)

The application sends a structured prompt to Gemini containing:
1. **Asset Metadata**: Name and Symbol.
2. **Current Pricing**: Live price and 24h percentage change.
3. **Technical Snapshot**: Current RSI value, MACD histogram, and SMA crossovers.
4. **Historical Context**: The last 5 price points to determine momentum.

Gemini returns a strictly typed JSON object based on a schema including a confidence score, a risk level, and 2-3 specific reasoning points.

## üìä Data Simulation (`services/marketData.ts`)

Since this is a predictive terminal, the `marketData` service implements a "Random Walk" volatility algorithm:
- **Base Volatility**: 0.1% per update.
- **History Generation**: Pre-populates 50 hours of historical data upon initialization.
- **Indicator Calculation**: Mathematically derives RSI and SMA from the generated history arrays.
